# SFI Courses

SFI consists of **four courses**: A, B, C, and D. Which courses a participant takes depends on their study path and starting level.

## Overview

| Course | Study Path(s) | Level | Swedish Equivalent |
|--------|--------------|-------|-------------------|
| **A** | Path 1 | Beginner | Pre-basic |
| **B** | Path 1, 2 | Basic | Approximately grade 3-4 |
| **C** | Path 2, 3 | Intermediate | Approximately grade 5-6 |
| **D** | Path 3 | Advanced | Secondary school level |

## Course Sequence by Study Path

### Study Path 1
```
A → B
```

### Study Path 2
```
B → C
```

### Study Path 3
```
C → D
```

## Detailed Course Information

- [Course A](kurs-a.md) — Basic literacy and oral Swedish
- [Course B](kurs-b.md) — Basic Swedish (reading, writing, speaking)
- [Course C](kurs-c.md) — Intermediate Swedish
- [Course D](kurs-d.md) — Advanced Swedish, secondary school level

## Time Requirements

| Course | Study Path 1 | Study Path 2 | Study Path 3 |
|--------|-------------|--------------|--------------|
| A | ~50 weeks | — | — |
| B | ~50 weeks | ~35 weeks | — |
| C | — | ~35 weeks | ~25 weeks |
| D | — | — | ~25 weeks |

*Note: Duration varies based on individual learning pace and study intensity.*

## After Completing SFI

Course D corresponds to **Swedish as a Second Language at secondary school level** (Svenska som andraspråk, SAS). After SFI, you can continue with:

- **SAS Grund** (Basic secondary level Swedish)
- **SAS 1-3** (Upper secondary level Swedish)
- **Upper secondary education** (Gymnasium/Komvux)
- **University studies** (requires Swedish proficiency)
