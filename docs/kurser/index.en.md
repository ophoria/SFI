# SFI Courses

SFI consists of **four courses**: A, B, C, and D. Which courses a participant takes depends on which study path they belong to.

## Course Overview

| Course | Study Path | Level | Format |
|--------|------------|-------|--------|
| [Course A](kurs-a.en.md) | 1 | Beginner | Primarily oral |
| [Course B](kurs-b.en.md) | 1, 2 | Basic | Oral and written |
| [Course C](kurs-c.en.md) | 2, 3 | Intermediate | Oral and written |
| [Course D](kurs-d.en.md) | 3 | Advanced | Oral and written |

## Courses per Study Path

### Study Path 1
```
Course A → Course B
```

### Study Path 2
```
Course B → Course C
```

### Study Path 3
```
Course C → Course D
```

## Grading Scale

All courses use the grading scale:
- **G** — Pass
- **IG** — Fail

## National Exams

National exams exist for:
- Course B
- Course C  
- Course D

!!! info "Note"
    Course A is an oral course where no national exam is written.

## Continuation Opportunities

After completing SFI, participants can continue with:
- Swedish as a Second Language (SAS) at upper secondary level
- Vocational Swedish
- Other courses within komvux
- Vocational higher education programs
