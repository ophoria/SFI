# SFI-kurser

SFI består av **fyra kurser**: A, B, C och D. Vilka kurser en deltagare läser beror på vilken studieväg personen tillhör.

## Kursöversikt

| Kurs | Studieväg | Nivå | Form |
|------|-----------|------|------|
| [Kurs A](kurs-a.md) | 1 | Nybörjare | Huvudsakligen muntlig |
| [Kurs B](kurs-b.md) | 1, 2 | Grundläggande | Muntlig och skriftlig |
| [Kurs C](kurs-c.md) | 2, 3 | Medel | Muntlig och skriftlig |
| [Kurs D](kurs-d.md) | 3 | Avancerad | Muntlig och skriftlig |

## Kurser per studieväg

### Studieväg 1
```
Kurs A → Kurs B
```

### Studieväg 2
```
Kurs B → Kurs C
```

### Studieväg 3
```
Kurs C → Kurs D
```

## Betygsskala

Samtliga kurser använder betygsskalan:
- **G** — Godkänd
- **IG** — Icke godkänd

## Nationella prov

Nationella prov finns för:
- Kurs B
- Kurs C  
- Kurs D

!!! info "Obs"
    Kurs A är en muntlig kurs där inget nationellt prov skrivs.

## Fortsättningsmöjligheter

Efter avslutad SFI kan deltagare fortsätta med:
- Svenska som andraspråk (SAS) på gymnasial nivå
- Yrkessvenska
- Andra kurser inom komvux
- Yrkeshögskoleutbildningar
