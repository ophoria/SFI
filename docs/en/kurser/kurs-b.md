# Course B within SFI

Course B is the second course within **Study Path 1** and the first course within **Study Path 2**. It builds on basic skills and develops both oral and written Swedish.

## Target Group

Course B is for you if you:

### Study Path 1
- Have completed Course A, OR
- Have some basic literacy but need more practice

### Study Path 2
- Have 6-9 years of schooling
- Can read and write in your native language
- Have basic literacy skills

## Learning Objectives

After completing Course B, you should be able to:

### Reading
- Read simple texts (news articles, letters, instructions)
- Understand the main points of familiar texts
- Use reading strategies (skimming, scanning)

### Writing
- Write simple texts (emails, letters, forms)
- Use basic grammar correctly
- Express opinions and experiences in writing

### Speaking
- Participate in conversations about everyday topics
- Give short presentations
- Express needs, opinions, and feelings

### Listening
- Understand spoken Swedish in everyday situations
- Follow instructions and directions
- Understand the main points of simple audio

## Content Areas

| Area | Content |
|------|---------|
| **Grammar** | Verbs, nouns, adjectives, sentence structure |
| **Vocabulary** | Expanded everyday and work-related vocabulary |
| **Text Types** | Emails, letters, forms, simple stories |
| **Society** | Swedish work life, education system, culture |
| **Study Skills** | Note-taking, using dictionaries, learning strategies |

## Duration

- **Study Path 1:** Approximately 50 weeks
- **Study Path 2:** Approximately 35 weeks

## Assessment

Course B assessment includes:

- **Written assignments**
- **Oral presentations**
- **Reading comprehension tests**
- **Teacher assessment**

## After Course B

| Study Path | Next Course |
|------------|-------------|
| Path 1 | Course B → **Completed SFI** |
| Path 2 | Course B → **Course C** |
