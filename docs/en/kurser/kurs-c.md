# Course C within SFI

Course C is the third course within **Study Path 2** and the first course within **Study Path 3**. It represents intermediate-level Swedish.

## Target Group

Course C is for you if you:

### Study Path 2
- Have completed Course B
- Are progressing from basic to intermediate level

### Study Path 3
- Have 10+ years of schooling or university education
- Have strong study skills
- Can learn languages quickly

## Learning Objectives

After completing Course C, you should be able to:

### Reading
- Read longer texts (news articles, short stories, factual texts)
- Understand detailed information
- Identify main points and supporting details
- Read texts relevant to work and study

### Writing
- Write coherent texts (letters, reports, narratives)
- Use varied sentence structures
- Express complex ideas in writing
- Adapt writing to different purposes

### Speaking
- Participate actively in discussions
- Give structured presentations
- Express and justify opinions
- Use Swedish in professional contexts

### Listening
- Understand extended speech on familiar topics
- Follow lectures and presentations
- Understand Swedish media (TV, radio, podcasts)

## Content Areas

| Area | Content |
|------|---------|
| **Advanced Grammar** | Complex sentences, verb forms, conjunctions |
| **Expanded Vocabulary** | Academic and professional vocabulary |
| **Text Analysis** | Understanding text structure and purpose |
| **Swedish Society** | Politics, history, culture in depth |
| **Work & Study** | Professional communication, study techniques |

## Duration

- **Study Path 2:** Approximately 35 weeks
- **Study Path 3:** Approximately 25 weeks

## Assessment

Course C assessment includes:

- **Written essays and reports**
- **Oral presentations**
- **Reading comprehension tests**
- **Grammar and vocabulary tests**
- **Teacher assessment**

## After Course C

| Study Path | Next Course |
|------------|-------------|
| Path 2 | Course C → **Completed SFI** |
| Path 3 | Course C → **Course D** |
