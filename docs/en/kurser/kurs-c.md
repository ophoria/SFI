# Course C within SFI

Course C is the third course within **Study Path 2** and the first course within **Study Path 3**. This is an intermediate level course for persons with study experience.

## Target Group

Course C is for you who:
- Have completed Course B (Study Path 2)
- Have good study capacity and longer schooling (Study Path 3)
- Are highly educated

## Course Goals

After completing the course, you should be able to:

### Reading
- Read and understand longer texts
- Identify main and subordinate clauses
- Interpret different text types (news, information, fiction)
- Use reading strategies

### Writing
- Write coherent texts
- Use different text types (letters, stories, argumentative texts)
- Use varied grammar and vocabulary
- Revise and improve own texts

### Speaking
- Participate in discussions and conversations
- Present topics orally
- Argue and give counterarguments
- Adapt language to situation

### Listening
- Understand main content in longer speech
- Follow discussions
- Understand news and programs in media

## Teaching Format

Course C focuses on:
- Independent work
- Text analysis and interpretation
- Oral presentations
- Discussions and debates
- Use of different sources

## Content

### Themes
1. **Society and Culture** — News, culture, traditions
2. **Work and Education** — Labor market, further education
3. **Everyday Life** — Housing, economy, relationships
4. **History and Future** — Sweden's history, future plans

### Language Areas
- **Vocabulary**: Extended vocabulary for different contexts
- **Grammar**: Tenses (past, future), subordinate clauses, passive voice
- **Style**: Formal and informal language
- **Pronunciation**: Fluency, stress, intonation

## National Exams

!!! info "National exam exists"
    For Course C, there are national exams issued by the Swedish National Agency for Education.

### Parts of the Exam
- Reading comprehension with different text types
- Writing task with different genres
- Listening comprehension
- Oral interaction and presentation

## Assessment

### Grades
- **G** — Pass
- **IG** — Fail

### Assessment Criteria
- Ability to communicate in different situations
- Linguistic correctness
- Variation in language use
- Understanding of different text types

## Course Length

Typical study time: **200–400 hours**

Study Path 3 participants can often complete the course faster thanks to study experience.

## Continuation

After passing Course C:
- **Study Path 2**: Completed education
- **Study Path 3**: Continuation to **Course D**
