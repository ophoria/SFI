# Navigering i SFI-wikin

## Så här hittar du i wikin

### Huvudmeny
Använd **flikarna ovanför** för att navigera mellan huvudavsnitten:
- Översikt
- Studievägar
- Kurser
- Regelverk
- Historia

### Sidomeny
I **sidomenyn till vänster** hittar du:
- Underavsnitt för varje huvudavsnitt
- Underkategorier som kan expanderas
- Länkar till relaterade sidor

### Innehållsförteckning
Till **höger på varje sida** finns en innehållsförteckning med länkar till alla rubriker på sidan.

## Navigeringsverktyg

### Sökfunktion
Tryck på **S-knappen** eller klicka på sökikonen överst för att söka i hela wikin.

### Taggar
Använd [taggsidan](taggar.md) för att hitta sidor efter ämne.

### Alfabetisk index
Se [innehållsförteckningen](index-sida.md) för en alfabetisk lista över alla sidor.

### Senast uppdaterade
Sidor visar automatiskt när de senast uppdaterades längst ner.

## Wiki-struktur

```
SFI-wikin/
├── Översikt           # Grundläggande information om SFI
├── Studievägar       # De tre studievägarna
├── Kurser            # Kurs A, B, C, D
├── Regelverk         # Lagar och förordningar
├── Historia          # SFI:s utveckling
├── Betyg & Prov      # Bedömning och nationella prov
├── Källor            # Referenser och källhänvisningar
└── Taggar            # Ämnesöversikt
```
